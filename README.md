# dotfiles
Requires GNU Stow.
Vim: Requires Vim 8+/Neovim 0.5+
Emacs: Requires GNU Emacs 26.2+

## Summary
Emacs: Modified version of DOOM
Tmux: prefix key rebind to `C-a` and switch key rebind to `^A`
Vim: cust0m
shells: fish/bash
terminal: alacritty

## Installation
``` bash
$ stow vim
$ stow fish
etc.
```
